# Component Sourcing

Component selection should support iteration speed and field maintenance, not just nominal spec-sheet performance.

## Sourcing Heuristics

- prefer parts with multiple suppliers
- avoid housings and cartridges that lock the project into one brand
- record connector types, thread standards, and sealing materials
- note lead times for items that can block rebuilds

## Priority Categories

| Category | What to capture |
|----------|-----------------|
| Pumps | flow range, suction behavior, service kits, voltage |
| Filters and housings | micron rating, pressure drop, seal material, replacement cost |
| Sensors | interface type, calibration needs, ingress protection |
| Fittings and valves | thread standard, body material, service access |
| Power components | charging profile, fuse strategy, field replaceability |

## Vendor Notes

Keep sourcing notes practical:

- what arrived quickly
- what was packaged poorly
- what had misleading documentation
- what worked but was painful to service

The goal is not a shopping catalog. It is a memory system for future build decisions.
