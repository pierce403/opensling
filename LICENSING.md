# Licensing Intent

This repository currently ships with an Apache-2.0 root license file.

Project intent going forward:

- Documentation text, diagrams, and educational media are intended to be published under CC BY 4.0 unless a page says otherwise.
- Code, scripts, configuration, and fabrication design sources are intended to remain under Apache-2.0 unless a file says otherwise.

If a stricter or different license applies to a specific asset, that asset should say so explicitly in its file header or adjacent README.
